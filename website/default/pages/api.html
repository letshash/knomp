API
Please use the API responsibly.
  • /api/stats global pool stats
  • /api/blocks global block stats
  • /api/pool_stats - historical stats
  • /api/payments - payment history
  • /api/worker_stats?taddr - historical time per pool json
  • /api/live_stats - live stats (websocket)
There are also Insight API's available at: explorer.spaceworks.co/api and tspace.explorer.spaceworks.co/api.
